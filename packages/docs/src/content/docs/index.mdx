## Usage

Using Jimp is simple.
Load an image, manipulate it, and save it.

<Steps>

1. Load an image.

   ```ts
   import Jimp from "jimp";

   const image = await Jimp.read("test/image.png");
   ```

2. Manipulate the image.

   ```ts
   image.resize({ width: 100 });
   ```

3. Save the image.

   ```ts
   await image.write("test/output.png");
   ```

</Steps>
